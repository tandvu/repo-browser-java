# Creating a Desktop Shortcut

To create a desktop shortcut for easy access:

## Method 1: Right-click Shortcut
1. Right-click on `Start Repo Browser.bat`
2. Select "Send to" → "Desktop (create shortcut)"
3. A shortcut will appear on your desktop

## Method 2: Manual Shortcut
1. Right-click on your Desktop
2. Select "New" → "Shortcut"  
3. Browse to the location of `Start Repo Browser.bat`
4. Give it a name like "Repo Browser"
5. Click "Finish"

## Method 3: Copy to Desktop
Simply copy `Start Repo Browser.bat` to your Desktop folder for direct access.

## Recommended Files for Distribution

For sharing with others, include these files in a folder:
- `Start Repo Browser.bat` ← Main launcher
- `repo-browser.jar` ← The application  
- `pom.xml` ← Required for Maven
- `RUNNING.md` ← Instructions

Users just need to double-click `Start Repo Browser.bat` to run the application!
